# Omnibus Seating Strategist v4.2

An overly engineered, hyper-analytical guide to bus seating to avoid social interaction, moist surfaces, and biological hazards (Rebecca).

## 🚌 Core Logic
This project treats a standard city bus layout as a tactical combat zone. It provides a data-driven approach to choosing a seat based on:
- **Biological Threats:** Specifically "Rebecca," whose breath has been categorized as a Level 4 Biohazard.
- **Acoustic Danger:** Zones where yapping from driver or seniors is most prevalent.
- **Surface Integrity:** Identifying the "Brad Cluster" where spilled liquids are common.
- **Tactical Isolation:** Finding the "Golden Void" for maximum peace.

## 🚀 Deployment
1. Upload `index.html` to any web server or GitHub Pages.
2. Review the **Visual Interference Heatmap** before boarding your next commute.
3. If Seat 17 is taken, consider walking.

## 🛠 Features
- **ASCII Bus Diagram:** A high-fidelity terminal-style map of the bus interior.
- **Threat Table:** A comprehensive breakdown of seat-specific risks and mitigation strategies.
- **The Leaning Protocol:** Advanced physics for window-seaters.
